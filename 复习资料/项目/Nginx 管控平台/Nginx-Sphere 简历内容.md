## 1、简历项目说明

**项目简介**：Nginx-Sphere 作为一个企业级 Nginx 全生命周期管理平台，旨在解决传统 Nginx 运维中配置分散、变更风险高、缺乏统一管控等痛点。该平台支持虚拟机 (VM) 模式和容器化 (K8S) 模式两种部署形态，实现了从版本管理、自动化分发安装、配置管理、性能监控到服务化部署的全流程闭环管理。目前，该平台已在全公司测试与生产环境深度落地，稳定承载了多区域 5000+ Nginx 实例的全量精细化管控。

**项目架构**
- **整体架构**：系统采用“管控层、下发层、执行层”三层架构分离设计。管控层基于 Spring Boot 实现租户/运维双控制台；下发层通过 NATS 消息中间件实现指令的高并发、异步化分发；执行层由高性能 Rust Agent 组成，实现对 Nginx 实例的精细化管控。
- **VM 管控模式**：针对传统虚拟机环境，支持通过管控平台安装 Nginx Agent，实现对存量老旧 Nginx 系统的平滑收编与批量配置管理，解决了配置分散、手动操作风险大的问题。
- **K8S 服务化模式**： 深度对接 MWFE 底座平台，实现 Nginx 的云原生 Serverless 化升级。用户在购买界面一键下单，由 MWFE 联动 K8S 自动化创建 Pod，实例预置 Agent Sidecar，支持全生命周期托管与弹性扩缩容。

**技术栈**
主要技术栈： Java 21, Spring Boot 3, Rust (Tokio/Axum), NATS, Kubernetes, Helm, MariaDB, Redis, Redisson.
